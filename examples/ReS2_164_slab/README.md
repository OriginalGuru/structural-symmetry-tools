# Example: ReS2 slab (12-atom supercell)

Hexagonal ReS2 slab, 2x2x1 supercell (4 Re, 8 S, 12 atoms).
Space group P-3m1 (No. 164). Generated by phonopy.
Lattice parameters: a = b = 5.835 Ang, c = 27.405 Ang (includes vacuum).

## Files

| File | Description |
|---|---|
| `POSCAR` | 12-atom slab structure in Direct coordinates |
| `findsym_input.txt` | Reference FINDSYM input generated from POSCAR |
| `README.md` | This file |

`symmetry_basis` and `symmetry_list` are not yet included for this example.
`vasprun.xml` is not included (too large for the repository).

## Running findsym

```bash
cd examples/ReS2_164_slab

# Generate input and run in one step
python ../../scripts/run_findsym.py
# Expected: Space Group: 164  D3d-3  P-3m1

# Or step by step
python ../../scripts/poscar_to_findsym.py
findsym findsym_input.txt

# For a distorted structure with looser tolerance
python ../../scripts/run_findsym.py POSCAR_distorted --postol 0.01 --output-stem distorted
```

## Expected result

```
Space Group: 164  D3d-3  P-3m1
```

## Notes

- The large c lattice parameter (27.4 Ang) reflects the slab geometry with
  vacuum padding. FINDSYM correctly identifies the in-plane symmetry.

- The 1/3, 2/3 fractional coordinates are characteristic of the hexagonal
  sublattice of the S atoms above and below the Re plane.

- The non-orthogonal lattice (hexagonal, with a 120 degree angle) is handled
  correctly by `poscar_to_findsym.py` via the `!latticeBasisVectors` keyword,
  which accepts the full 3x3 matrix directly.
